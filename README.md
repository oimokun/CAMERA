# CAMERA　　

### 概要
カメラを起動して写真を撮影、またはフォトライブラリーから写真を選択した後エフェクトをかけることができます。  
その写真をSNSに投稿も可能です。  
  
### 制作背景
スマホカメラという誰もが使う機能をより便利なものにしたいと考えて取り掛かりました。  
多くのユーザーは一つのカメラアプリのみを使用するのではなく、数多くあるアプリを状況によって使い分けています。  
そこで一つでも他にはない機能がついていたら使用される可能性があるのではと思い、制作しました。
